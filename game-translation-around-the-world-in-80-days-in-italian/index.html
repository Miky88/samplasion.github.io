Game Translation: Around the World in 80 Days in Italian

I've made an Italian version of the game Around the World in 80 Days. It contains the original files from the Italian Big Fish demo.

Screenshots

Downloads

Windows

  • Zip File
  • Installer

macOS

  • .app Folder (for Big Fish Games)
  • Resources Folder (for Playrix)

Installation Tutorials

Windows

Zip File

  1. Extract the contents of the Around the World in 80 Days ITA.zip file.
  2. Go where you extracted it, create a shortcut to the game.exe file and put it on the desktop.

Installer

  1. Run the Around the World in 80 Days.exe installer.
  2. After finishing to install, go where you installed the game, create a shortcut to the game.exe file and put it on the desktop.

Mac

.app Folder (for Big Fish Games)

  1. Locate the game Around the World in 80 Days in /Applications.
  2. Right click the game's icon and choose Show Package Contents.
  3. Copy the file/folder Around the World in 80 Days.app into Contents/Resources and replace the original.

Resources Folder (for Playrix)

  1. Locate the game Around the World in 80 Days in /Applications.
  2. Right click the game's icon and choose Show Package Contents.
  3. Copy the Resources folder into Contents and replace the original.
