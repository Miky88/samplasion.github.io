Welcome, Ghost!

A brief description of the new blog backend.

By Samplasion 5 days ago in Meta

I've decided I wanted to change the blog's backend to something fresher and newer. That's why I've decided to switch to Ghost, an open-source, Node-powered, headless blog backend. It's got a full-fledged admin interface, out-of-the-box theme integration (some are paid, but eh), integrations and a web-based post maker (which is what I originally wanted to create for the old framework).
That's why I can say I'm particularly happy with this. One thing I should say is: if you want to do something like this on GH Pages, you'll have to follow an online guide with static site mirroring from the computer. But that's a minor annoyance.
